# Contributing to Portfolio Website

Thank you for your interest in contributing! Please follow these guidelines:

## Code Style
- Use ESLint with the provided configuration.
- Use Prettier for formatting.
- Follow React and JavaScript best practices.

## Branching
- Create feature branches from `main`.
- Use descriptive branch names.

## Pull Requests
- Ensure tests pass before submitting a PR.
- Include descriptive titles and detailed descriptions.

## Reporting Issues
- Use GitHub issues for reporting bugs or feature requests.
- Provide steps to reproduce and screenshots when applicable.

## Code of Conduct
- Please read CODE_OF_CONDUCT.md

We appreciate your contributions!
